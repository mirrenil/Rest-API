# Rest-API with CRUD using Node.js and Express

```
# Routes
GET      /api/products
POST     /api/products
GET      /api/products/:id
PUT      /api/products/:id
DELETE   /api/products/:id

```

## Usage

```
# Install dependencies
npm install
yarn install


# Run in production
npm start
yarn start
```
